FASTEST WAY TO PUT THIS ONLINE WITH VERCEL

1. Go to vercel.com.
2. Create a free account.
3. Click Add New Project.
4. Choose one of these methods:

METHOD A: Upload through GitHub
- Create a free GitHub account.
- Create a new repository.
- Upload all files from this folder.
- In Vercel, import the GitHub repository.
- Click Deploy.

METHOD B: Use Vercel CLI
- Install Node.js.
- Install Vercel CLI with: npm i -g vercel
- Open this folder in Terminal.
- Run: vercel
- Follow the prompts.
- Run: vercel --prod when ready.

CONNECT VEYROXAUTO.COM TO VERCEL

1. In Vercel, open your project.
2. Go to Settings.
3. Go to Domains.
4. Add VeyroxAuto.com.
5. Vercel will show DNS records.
6. Go to GoDaddy.
7. Open your VeyroxAuto.com domain.
8. Go to DNS settings.
9. Add or update the records Vercel gives you.
10. Wait for DNS to update.
